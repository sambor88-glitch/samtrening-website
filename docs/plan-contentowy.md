# Plan contentowy SAMtrening 2026

Pełny plan znajduje się w oryginalnym dokumencie `Plan_contentowy_samtrening.docx`. Ten plik to skrócona wersja dla repo — żeby łatwo sprawdzić, co kiedy ma być opublikowane.

## Zasady ogólne

Każdy wpis musi zawierać:
- Pełne imię autora (Kasia Samborska / Maciej Samborski / Bartosz Świerk)
- Zdjęcie autora (nie stockowe)
- Minimum 1 link wewnętrzny do strony ofertowej
- Lokalne słowo kluczowe ("Kraków", "Stare Miasto", "Plac Na Groblach") wplecione 2-3 razy
- Meta description max 155 znaków z CTA
- Slug URL bez polskich znaków, bez emoji, bez dat
